## 📚 Sobre o Projeto
Este arquivo HTML é parte dos meus estudos sobre como **incorporar imagens** em páginas web. Aprendi a usar imagens locais (salvas no projeto), imagens externas (hospedadas na web) e até GIFs animados! Além disso, explorei o fundo colorido com hexadecimal.

## 📁 Arquivo
| Arquivo | Descrição |
|---------|-----------|
| `imagem.html` | Demonstra diferentes formas de adicionar imagens em uma página HTML |

## 🎯 O que aprendi

### Tag `<img>` - Principais atributos:
- ✅ **`src`** (source) - Caminho da imagem
- ✅ **`alt`** (alternative text) - Texto alternativo (acessibilidade)
- ✅ Imagens **locais** (dentro da pasta do projeto)
- ✅ Imagens **externas** (links da internet)
- ✅ **GIFs animados** como imagens
- ✅ Imagens com **fundo transparente** (PNG)
